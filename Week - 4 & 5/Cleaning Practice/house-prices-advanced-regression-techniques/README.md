# Performing LR On House prices data 

## Understanding The Features of the columns 

1. MSSUBClASS -> tells about the No. of Floor & design is Old or New 
2. MSZoning -> Area Zone Acc. to Gov.
3. LotFrontage -> Road se ghar kitni width me laga hai
4. LotArea -> Total Land Area
5. Stret -> Type of Roal near House
    - Pave -> Pakki road (cement/asphalt)
    - Grvl ->  Kankar/mitti wali road
6. Alley -> aju baju mai choti road ya h nhi ? ya Continously house he h ?
7. LotShape -> Shape of Land (Reg-> Proper Rectangle)
8. Utilities -> Basic Needs like
    - AllPub =>	Bijli + Pani + Gas + Sewer
    - NoSewr =>	Sewer nahi
    - ELO =>	Sirf bijli
9. Neighborhood -> ghar Kis area mai H (Name of the area)
10. OverallQual -> Overall Ghar Kitna Acha H ?
11. GrLivArea -> Ghar ke Andar kitni jagah H ? 
    - Living area (ground + upper floors)
    - Basement count nhi hota
12. LandContour -> Defines elevation or height of height using imaginary lines
13. LandSlope -> Land Flat h ya banked h (chadhai ya slope jaisa h)
